## 📋 Sobre o projeto

Projeto demonstrativo de landing page para clínica odontológica fictícia localizada em São Gabriel, RS. Desenvolvido com HTML, CSS e JavaScript puro — sem frameworks ou dependências externas.

O objetivo foi criar uma página profissional, responsiva e de carregamento rápido, adequada para clínicas e consultórios que desejam presença digital.

---

## ✨ Funcionalidades

- **Nav fixo** com efeito glassmorphism ao rolar a página
- **Menu hamburger** funcional para dispositivos móveis
- **Seção Hero** com imagem de fundo, overlay e animações de entrada
- **Grid de serviços** com 6 cards e efeito hover com barra animada
- **Seção Equipe** com 3 dentistas em grid estático
- **Seção Sobre** com estatísticas e frame decorativo
- **Depoimentos** com glassmorphism sobre fundo verde-água
- **CTA de agendamento** com botão direto para WhatsApp
- **Scroll reveal** — elementos surgem suavemente ao entrar na tela
- **100% responsivo** — mobile, tablet e desktop

## 🚀 Como rodar

Sem instalação necessária. Basta clonar o repositório e abrir o `index.html` no browser:

```bash
git clone https://github.com/seu-usuario/clinica-oral-sg.git
cd clinica-oral-sg
```

Ou usar a extensão **Live Server** no VS Code para visualização com hot reload.
